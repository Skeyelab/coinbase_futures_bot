# Issue Tracker

**Type:** GitHub Issues
**Repo:** Skeyelab/coinbase_futures_bot
**CLI:** `gh`

## Creating issues

```bash
gh issue create --title "..." --body "..."
```

## Listing issues

```bash
gh issue list
```

## Closing issues

```bash
gh issue close <number>
```

## Labelling

```bash
gh issue edit <number> --add-label "<label>"
```
